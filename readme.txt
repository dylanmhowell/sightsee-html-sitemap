An easy HTML Sitemap plugin created by the team at Sightsee Design.

== Description ==

Insert this block into a new page to create your own HTML sitemap. We like to link to this HTML sitemap from our footer. This has shown to increase crawling, indexing, and rankings across many of our projects.

== Installation ==

1. Install the plugin through the WordPress plugins screen directly by either uploading the zip file or searching the repository.

2. Activate the plugin through the 'Plugins' screen in WordPress.

3. Add the new "HTML Sitemap" Block to a new page.


== Frequently Asked Questions ==

= What does a HTML Sitemap do? =

It is simply a list of all pages and posts on your website. This can be helpful for larger or more complex sites that might have less than optimal internal link structures.
